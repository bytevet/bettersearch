# Chrome Web Store Listing

## Summary (≤132 chars)

Fuzzy full-text search for any web page. Find and highlight text instantly with typo tolerance, prefix matching, and ranked results.

## Detailed Description

BetterSearch brings powerful full-text search to every web page you visit.

Chrome's built-in Ctrl+F only matches exact text. BetterSearch goes further — it indexes the entire page and lets you search with fuzzy matching, so typos and partial words still find what you're looking for.

𝗛𝗼𝘄 𝗶𝘁 𝘄𝗼𝗿𝗸𝘀
1. Click the BetterSearch icon, press Alt+F, or right-click and select "Search with BetterSearch"
2. A side panel opens — type your query
3. Results appear instantly, ranked by relevance
4. Click any result to scroll to it and see the matching terms highlighted on the page

𝗙𝗲𝗮𝘁𝘂𝗿𝗲𝘀
• Fuzzy search — finds results even with typos or partial words
• Prefix matching — start typing and results appear immediately
• Ranked results — most relevant matches shown first
• Highlighted matches — matching terms are highlighted directly on the page
• Side panel — stays open alongside the page without blocking content
• Dark mode — switch between light, dark, or system theme
• Keyboard shortcut — Alt+F to open or close instantly
• Right-click menu — search from the context menu on any page
• SPA support — automatically re-indexes when page content changes

𝗣𝗿𝗶𝘃𝗮𝗰𝘆
BetterSearch processes everything locally in your browser. No data is sent to any server. The extension does not collect, store, or transmit any personal information or browsing data.

𝗣𝗲𝗿𝗺𝗶𝘀𝘀𝗶𝗼𝗻𝘀 — 𝗼𝗻𝗹𝘆 𝘄𝗵𝗮𝘁'𝘀 𝗻𝗲𝗲𝗱𝗲𝗱
• activeTab — reads the current page's content for indexing, only when you activate the extension
• scripting — injects the search script into the active tab on demand
• sidePanel — displays the search interface beside the page
• contextMenus — adds "Search with BetterSearch" to the right-click menu

No host permissions. No background access to your browsing. The extension only runs when you explicitly activate it.
